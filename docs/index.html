PSVita Fast Post FX

This post processing stack is optimized for performance, particularly on modern systems. It can use either a single or dual pass to apply various effects to the image. These effects are applied using the vertex shader rather than the fragment (pixel) shader, which helps to boost performance. The stack uses a tesselated quad instead of the screen buffer because this allows for more flexibility in applying effects such as vignetting, which would be difficult to achieve with only four vertices.

The single shader included in this stack offers a range of effects, including gaussian blur, depth based outline, normal mapping screen warping, tinted vignette, ramp based color grading, brightness and contrast control, film grain, color space conversion, depth based fog, sun aura, height based fog & bloom (note: bloom may not work as intended).

Techniques Used

Before the image is displayed, the game view, depth, height, and normal values are first rendered to two internal buffers (render textures). These buffers are then used to apply the post processing effects using the vertex shader. This approach provides a significant performance boost with minimal performance penalty. The tesselated quad is placed in front of the camera that renders the image to the screen.

Features

Features

Gaussian Blur

This effect adds blur to the image using a Gaussian function. It has a light performance penalty, but the performance has been fine-tuned in this shader.

Depth Based Outline

This effect adds an outline to objects based on their depth in the scene. It has a medium performance penalty, and the quality may suffer if it is further optimized. However, the performance has been fine-tuned in this shader as much as possible without sacrificing quality.

Normal Mapping Screen Warping

This effect warps the screen based on the normals of the objects in the scene. It has a light performance penalty and the performance has been fine-tuned in this shader.

Tinted Vignette

This effect adds a vignette around the edges of the image, with a customizable color. It has a light performance penalty and the performance has been fine-tuned in this shader.

Ramp Based Color Grading

This effect allows you to adjust the color balance of the image using a color ramp. It has a light performance penalty and the performance has been fine-tuned in this shader.

Brightness and Contrast Control

This effect allows you to adjust the brightness and contrast of the image. It has a light performance penalty and the performance has been fine-tuned in this shader.

Film Grain

This effect adds film grain to the image for a more cinematic look. It has a light performance penalty and the performance has been fine-tuned in this shader.

Color Space Conversion

This effect allows you to convert the image from one color space to another (e.g. RGB to HSL). It has a light performance penalty and the performance has been fine-tuned in this shader.

Depth Based Fog

This effect adds fog to the image based on the depth of the objects in the scene. It has a medium performance penalty and the performance has been fine-tuned in this shader.

Sun Aura

This effect adds an aura around a bright light source (e.g. the sun). It has a medium performance penalty and the performance has been fine-tuned in this shader.

Height Based Fog

This effect adds fog to the image based on the height of the objects in the scene. It has a medium performance penalty and the performance has been fine-tuned in this shader.

Bloom

This effect adds a bloom effect to the image, where bright areas of the image appear to "glow." However, note: bloom may not work as intended in this shader. It has a heavy performance penalty and the performance has not been fine-tuned in this shader.

Debug Views

This shader includes a number of debug views to help troubleshoot and understand the inner workings of the post processing stack. These views allow you to visualize the data in the render textures (e.g. game view, depth, height, normal) and can be useful for debugging and optimizing the effects. It has a light performance penalty and the performance has been fine-tuned in this shader.
